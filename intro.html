Introdução


Javascript é uma linguagem de programação que é bastante utilizada na web para acrescentar interatividade à um site, como comportamentos quando, por exemplo, você clica em um botão, animações, estilos dinâmicos de sites, etc, mas também, se você tiver experiência com a linguagem, é possível utilizá-la na criação de jogos, gráficos 2D e 3D animados, aplicativos e muito mais, já que é uma linguagem dinâmica e relativamente simples de aprender.

Neste tutorial, você vai aprender o básico de Javascript e Javascript DOM (Document Object Model), que é a parte do Javascript que permite você fazer alterações na página web.
